Training Pipeline Roadmap

  • 1. Extract Frames

    Extract exact, timestamped image frames from the raw video footage.

  • 2. AI Auto-Labeling

    Automatically detect and draw bounding boxes around road surfaces and hazards using AI.

  • 3. Manual Adjustments

    Review the AI's work and manually fix or add labels for difficult edge cases using the canvas.

  • 4. Data Synchronization

    Merge the visual labels with the bicycle's physical vibration sensor data based on timestamps.

  • 5. Model Training

    Train the machine learning model on the combined dataset to recognize road surfaces.

  • 6. Live Testing

    Run the trained model and view live, real-time predictions directly on the dashboard map.

Next-Generation Capabilities

Label Fusion

Automatically snap visual annotations onto exact IMU vibrations. Seamlessly fuse your camera-based ground truth with high-frequency telemetry data to build a unified training dataset.

Temporal Logic & Sync

Precisely align asynchronous sensors using a customizable merge tolerance. Intelligently manage time gaps using Forward-Fill or Linear Interpolation strategies to ensure data continuity.

Quarter-Car DSP

Apply advanced Digital Signal Processing to cut road resonance from the bicycle's frame bounce. Mathematically isolate the true, high-frequency road surface texture from generic movement.

Visual Latent Vectors

Enhance the machine learning model's predictive power by feeding it camera-based latent parameters. Let the AI visually anticipate upcoming terrain before the wheels ever touch it.

6-DoF Vibration Sequences

Capture the full physical response of the bicycle across 6 Degrees of Freedom. Analyze multi-axis acceleration and gyroscope sequences over time to identify distinct road signatures.
